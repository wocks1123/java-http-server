# Java HTTP Server

Java 소켓 프로그래밍을 통해 HTTP 웹 서버를 직접 구현하는 학습 프로젝트

## 학습 목표

- HTTP 프로토콜의 동작 원리 이해
- 서블릿 컨테이너가 하는 일을 직접 구현하며 체득

## 기술 스택

- Java 21
- Gradle (Kotlin DSL)
- JUnit 5

## 실행 방법

```bash
# 빌드
./gradlew build

# 실행
./gradlew run

# 테스트
./gradlew test
```

서버 실행 후 브라우저에서 `http://localhost:8080` 접속
